* Buscar utilizando google dork:
python3 sql.py --dork [palavras]
 
 * Ativando função de SCAN:
 python3 sql.py --dork [palavras] --scan

 * Com Proxy:
 python3 sql.py --dork [palavras] --proxy 0.0.0.0:1337
